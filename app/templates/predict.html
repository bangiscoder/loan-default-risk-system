3MTT Mentorship Project

Loan Default Risk Prediction System

A machine learning-powered web application that estimates the likelihood of loan default based on borrower information.

Developer: Babangida Yohanna

Built as a 3MTT Mentorship Project in collaboration between 3MTT Nigeria and WESOnline.

This MVP uses a public credit dataset for demonstration and will later be improved for Nigerian microcredit realities.
